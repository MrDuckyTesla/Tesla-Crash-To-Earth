Controls (Currently):

General:
  Swap Character States:   Enter
  Change Character Color:  C
  Reset Character Color:   R
OverWorld:
  Move Up:                 W, ArrowUp
  Move Left:               A, ArrowLeft
  Move Down:               S, ArrowDown
  Move Right:              D, ArrowRight
  Sprint:                  Shift
  Attack:                  X
Battle:
  Move Left:               A, ArrowLeft
  Move Right:              D, ArrowRight
  Jump:                    Space
  Fast Fall:               S, ArrowDown
  Dash:                    Shift
